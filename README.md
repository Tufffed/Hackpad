# Tuffed's Macropad
This has 5 MX keys with an additional Rotary Encoder, an OLED display, and RGB.

## CAD
The case it built with 4 M3 screws and 4 matching HeatPress inserts. The case consits of a bottom shell and a top lid. The PCB is held in place as it's screwed with another 3 M3 screws with the same heat press inserts to match.
